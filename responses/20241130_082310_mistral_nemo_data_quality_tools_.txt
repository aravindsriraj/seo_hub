Data quality tools are software solutions designed to help organizations ensure the accuracy, consistency, and completeness of their data. Here are some key data quality tools and their functionalities:

1. **DataProfiling Tools**: These tools help understand the structure, content, and quality of data.
   - **Talend Data Profiling**: Offers fast data profiling with automated data quality checks.
   - **Trifacta Wrangler**: Helps clean, transform, and enrich data using a user-friendly interface.
   - **Pentaho Data Integration**: Provides data profiling along with ETL (Extract, Transform, Load) capabilities.

2. **Data Cleansing Tools**: These tools help clean and standardize data.
   - **SAS Data Management**: Offers data cleansing, matching, and standardization features.
   - **Informatica Data Quality**: Cleanses, standardizes, and enriches data across different platforms.
   - **Zoho DataPrep**: Allows easy data cleansing, normalization, and transformation.

3. **Master Data Management (MDM) Tools**: These tools help manage and govern master data (like customer, product, or location data).
   - **SAP Master Data Governance**: Part of SAP's ERP systems, it provides MDM capabilities.
   - **IBM Master Data Management**: Centers on customer, product, location, and reference data.
   - **Microsoft Master Data Services**: Integrated with SQL Server and SharePoint for managing master data.

4. **Data Quality Monitoring Tools**: These tools continuously monitor data quality post-cleansing or integration.
   - **Google Cloud Data Quality**: Monitors and validates data quality with automated checks.
   - **Microsoft Azure Data Catalog**: Enables data discovery, description, and lineage, helping maintain data quality.
   - **Amazon Data Lake**: Allows organizations to monitor data quality in their data lakes.

5. **Data Governance Tools**: These tools help manage and control data quality, ensuring compliance with policies.
   - **ER/Studio**: A data modeling and database design tool supporting data governance.
   - **Collibra Data Governance Center**: Offers data governance capabilities like data lineage, data cataloging, and data quality management.
   - **IBM Data Governance on Cloud**: Provides data governance features with built-in compliance and security controls.

6. **Open-Source Data Quality Tools**: These tools offer data quality functionalities for free or under open-source licenses.
   - **OpenRefine**: A powerful tool for working with messy data, supporting cleaning and transforming data.
   - **Apache Atlas**: Provides data governance, metadata management, and data lineage capabilities.
